# GAINAX-BP4-tools

scripts for converting bp4 files found inside of Shinji Ikari Raising Project (新世紀エヴァンゲリオン 碇シンジ育成計画) 
these are mostly for UI elements, I'm sure other more recent GAINAX games may use this file format as well

usage:

python bp42png.py input.bp4 output.png

python png2bp4.py original.bp4 edited.png output.bp4

uses original bp4 file as a sort of template for constructing a newly edited bp4 file
